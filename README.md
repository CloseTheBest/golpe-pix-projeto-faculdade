# Projeto - Prevenção ao Golpe do Pix (Engenharia Social)

Este repositório contém os arquivos do projeto acadêmico desenvolvido para propor uma solução tecnológica de mitigação ao golpe do Pix baseado em engenharia social.

## Estrutura do Repositório

- **/artigo/**  
  Contém o arquivo .pdf do artigo científico.

- **/codigo/**  
  Exemplos de código em Java ilustrando partes da solução (API, modelos, lógica).

- **/diagramas/**  
  Diagramas ilustrativos da arquitetura e banco de dados.

## Objetivo do Projeto

O projeto propõe uma solução distribuída capaz de identificar padrões suspeitos de transações Pix, enviar alertas, registrar tentativas suspeitas e auxiliar na prevenção ao golpe aplicado por engenharia social.
